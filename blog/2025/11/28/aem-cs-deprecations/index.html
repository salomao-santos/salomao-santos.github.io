AEM 2025: Deprecations and Breaking Changes

Important deprecations in AEM Cloud Service: Java 11 EOL, deprecated APIs, logging policy changes

November 28, 2025 · By Salomao da Silva Santos

⚠️ Depreciações e Remoções

Java 11 Runtime - DEPRECIADO

Migração obrigatória para Java 21

Timeline:

  • 18 de Setembro de 2025: Upgrade de ambientes Dev e RDE
  • 14 de Outubro de 2025: Upgrade de Stage e Production

Ações Necessárias:

  • Validar site e processos em Dev/RDE
  • Resolver dependências incompatíveis com Java 21
  • Verificar requisitos do Java 21

APIs Java Depreciadas - Remoção em 31 de Agosto

Remova uso dessas APIs imediatamente

APIs Marcadas para Remoção:

  • org.apache.sling.commons.auth
  • org.apache.felix.webconsole
  • org.eclipse.jetty
  • com.mongodb
  • org.apache.abdera
  • org.apache.felix.http.whiteboard
  • org.apache.cocoon.xml
  • ch.qos.logback
  • org.slf4j.spi e org.slf4j.event
  • org.apache.log4j
  • com.google.common
  • com.drew
  • org.bson
  • org.apache.jackrabbit.oak.plugins.blob
  • org.apache.jackrabbit.oak.plugins.memory

Notificações: Actions Center alertará sobre uso detectado. Após 13 de novembro, avisos aparecerão em builds do Cloud Manager.

Política de Configuração de Logs Java - Enforcement

A partir de 30 de Outubro: Configurações customizadas serão ignoradas

Mudanças:

  • Logs devem seguir formato padrão AEM
  • Configurações customizadas (formato, arquivos de saída, níveis de log) não são mais suportadas
  • Logs devem permanecer nos arquivos padrão
  • Níveis de log padrão para código AEM devem ser preservados

Impactos Potenciais:

  • Sistemas de log forwarding podem precisar ajustar regras de ingestão
  • Reverter para níveis padrão pode aumentar volume de logs
  • Processos downstream que dependem de formato customizado precisam ser atualizados
